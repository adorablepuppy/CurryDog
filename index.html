CurryDog by adorablepuppy

A game engine written in Haskell that will provide 2d and 3d physics, a sound framework, and much more.

CurryDog aims to provide a complete suite of functions for 2d and 3d game development. These functions will be organized into modules so users may pick and choose what functions they want to include in their projects.

Dependencies

sdl, sdl-image, sdl-ttf, sdl-mixer

Install

You must first install [SDL][1], [SDL-image][2], [SDL-ttf][3], and [SDL-mixer][4] through cabal. To do this, you must have a the SDL packages themselves from [libsdl.org][5]. On Linux, this should be trivial. On Windows, it will be a bit of a trial. If you use Windows, I suggest you follow this tutorial: [A Haskell Aventure In Windows][6] [1]: http://hackage.haskell.org/package/SDL-0.6.2 "SDL on Hackage" [2]: http://hackage.haskell.org/package/SDL-image "SDL-image on Hackage" [3]: http://hackage.haskell.org/package/SDL-ttf "SDL-ttf on Hackage" [4]: http://hackage.haskell.org/package/SDL-mixer "SDL-mixer on Hackage" [5]: http://www.libsdl.org/ "libsdl" [6]: http://www.animal-machine.com/blog/2010/04/a-haskell-adventure-in-windows/ "A Haskell Aventure In Windows" Next, just do a _git clone git://github.com/adorablepuppy/CurryDog.git_ To build with CurryDog, select each module you wish to use and include them in your project. For example: _ghc MyGame.lhs CurryDog.lhs CurryDog.Data.lhs_ CurryDog will include a way to inject your main function into the game loop, allowing your program to contain the Main module.
